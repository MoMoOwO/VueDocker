# Vue-Docker

> 基于 Vue 打造的 Docker app

## 技术支持

### 项目安装

1. 项目下载：`git clone https://github.com/MoMoOwO/VueDocker.git`
2. 安装依赖：`npm install`

### 项目运行

1. 开发运行：`npm run serve`
2. 项目构建：`npm run build`

## 开发依赖

1. [Vue.js](https://cn.vuejs.org/)
2. [Element UI](https://element.eleme.cn/#/zh-CN)
